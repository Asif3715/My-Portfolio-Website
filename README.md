# Asif Khan — Portfolio

Personal portfolio website built with Next.js, Framer Motion, and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion, GSAP, Lenis
- **Language:** JavaScript

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```

## Pages

- **Home** — Hero section with animated name slider
- **About** — Education, experience, and skills
- **Projects** — Selected work with GitHub repo previews
- **Contact** — Contact form and details

## Deployment

Deployed on [Vercel](https://portfolio4-pearl.vercel.app).
